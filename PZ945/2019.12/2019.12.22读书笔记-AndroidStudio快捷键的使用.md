# AndroidStudio 中快捷键的使用
 1. 参数提示 Ctrl + P

 2. 自动提取方法 Ctrl + Alt + M

 3. 自动提取参数 Ctrl + Alt + P

 4. 自动生成变量 Ctrl + Alt + V

 5. 搜索某个方法是否被调用 Alt + F7

 6. 重构快捷键 Ctrl + Alt + Shift + T

 7. 包裹代码 Ctrl + Alt + T

 8. 批量重命名 shift + F6

 9. 查看方法调用层次结构 Ctrl + Alt + H

 10. 查看类的层次结构 Ctrl + H

 11. 错误快捷提示 Alt + Enter

 12. 显示当前类所有的方法 Ctrl + F12

 13. 变量高亮 Ctrl + Shift + F7

 14. 快速跳转到父类 Ctrl + U

 15. 快速跳转到实现 Ctrl + Alt + B
 
 16. 覆写或者实现方法 Ctrl + O

 17. 生成默认方法 Alt+Insert
